## 使用

[PC端预览地址](https://www.hduzplus.xyz/)

直接体验，复制以下代码到浏览器控制台内：
```
const script = document.createElement('script');
script.src = 'https://static.hduzplus.xyz/Marisa/index.js';
document.body.appendChild(script);
```

操控方式：
* 方向：上下左右键
* 加速移动：按住a，同时按住左/右键
* 得意的笑：按住s
